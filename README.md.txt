# Site officiel UESBA

## But du repo
Ce repo contient la structure initiale du site officiel de l'UESBA pour que l'équipe puisse commencer à travailler.

## Installation locale
1. Installer WampServer.
2. Créer la base de données `UESBA` via phpMyAdmin.
3. Modifier `config/config.php` si nécessaire pour correspondre aux identifiants locaux.
4. Placer le dossier `uesba-site` dans `www` (Wamp).
5. Accéder à `http://localhost/uesba-site/` pour tester.

## Contenu du repo
- `index.php` : page d'accueil
- `css/` : dossier pour styles
- `js/` : dossier pour scripts
- `images/` : dossier pour images
- `config/config.php` : connexion à la base
- `sql/uesba.sql` : export initial de la base (optionnel)
